## Installation

### From Snap Store

To install Arturo from the Snap Store, simply run:
```bash
sudo snap install arturo
```

### From Source

You can also build and install the snap locally:
```bash
git clone https://github.com/arturo-lang/snapcraft.git
cd snapcraft
snapcraft
sudo snap install arturo_*.snap --dangerous
```
